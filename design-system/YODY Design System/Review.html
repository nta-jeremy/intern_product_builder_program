Đánh giá toàn diện về YODY Design System hiện tại (v2026.05 · Light Hightech) — điều gì đang hoạt động tốt, ba rủi ro cấu trúc cần xử lý trước v2, và một lộ trình ba pha để mở rộng system từ một portal nội bộ sang một brand identity đầy đủ cho YODY.

Kết luận tổng quan
Foundation rất chắc. Token discipline, content tone, và 9 nguyên tắc enforcement đều ở mức trên trung bình ngành. Nhưng system đang gắn quá chặt vào một narrative (ITDX × IPO 2030) và chưa phục vụ thương hiệu YODY tiêu dùng. Ba vấn đề cấu trúc — font sprawl, logo divergence, scope gap — cần khoá lại trước khi xây thêm UI kit thứ hai.

03 · structural risks

Ba rủi ro cần khoá trước v2

Ưu tiên theo blast radius. P0 = fix trước khi viết thêm component. P1 = fix trước khi build UI kit thứ hai. P2 = nâng cấp sạch.

I

Scope gap — system chỉ phục vụ ITDX, không phục vụ YODY thương hiệu

NowYODY là một fashion retail brand với 274 cửa hàng, website ecommerce yody.vn, app mua hàng, in-store signage, lookbook marketing. Hiện tại system chỉ cover 1 surface: internal EA portal. Mọi token, mọi rule, mọi UI kit đều B2B / executive / projector-readable. Không có thứ gì áp được lên product card thương mại, mobile checkout, retail POS.

FixTách thành 3 layers: (1) Foundations — logo, color, type, motion (dùng cho mọi surface). (2) ITDX library — KPI, domain, insight, panel (hiện tại). (3) Commerce library — product card, size selector, cart, lookbook hero (build mới). Rename root README để scope rõ. Add ui_kits/yody-commerce/ như đợt 2.

P0 · Critical Effort · 3w
II

Font sprawl — 6 families, system tự nói "no font #6"

NowInter + Be Vietnam Pro + Montserrat + Playfair Display + JetBrains Mono + Dancing Script. DESIGN.md §3.2 viết "No font #6. If you reach for a sixth family, you're trying to compensate for a layout problem." — nhưng system đã có 6. Inter và Be Vietnam Pro overlap (cả hai đều modern sans + Vietnamese capable). Dancing Script "hero signature" chưa được dùng ở đâu trong UI kit.

FixDrop xuống 4 families: Be Vietnam Pro (body + brand chrome — thay luôn Inter, đỡ phải care Vietnamese subset), Montserrat (display + numerals), Playfair italic (1 hero moment), JetBrains Mono (technical). Drop Inter và Dancing Script. Tiết kiệm ~80KB font CSS payload và một đường conflict trong rule book.

P0 · Critical Effort · 1w
III

Logo divergence — lockup hệ thống không nối với master mark

NowMaster logo YODY là navy "yo" + white "dy" trên square gold — rất đặc trưng, rất nhận diện. System invent một lockup mới: tile 28×28 gradient navy → iris với chữ "y" trắng + wordmark Quicksand "yody·id". Hai identity competing trong cùng một brand. Người xem portal không link được với cửa hàng / website / app.

FixGround tất cả lockup vào master logo. Tile header dùng gold square + navy "y" (cùng DNA với storefront). Wordmark dùng đúng "yody" letterform của logo, không phải Quicksand suy diễn. Sub-mark "·id" giữ được như product-line modifier nhưng dùng family chính của system, không phải invent mới.

P0 · Critical Effort · 4d
IV

Source-of-truth conflict — README mâu thuẫn với DESIGN.md

NowREADME mô tả system là "Inter-led" và "BUILD = navy". DESIGN.md mô tả "5-font composition với Be Vietnam Pro làm brand chrome" và "BUILD = iris #7c6cf5". Hai docs lệch nhau ở tầng định nghĩa, không phải tầng diễn giải.

FixChọn DESIGN.md làm canonical (đã có ghi chú "this file wins" ở cuối). Rewrite README §VISUAL FOUNDATIONS để khớp: "Be Vietnam Pro-led, with Montserrat for display moments. BUILD = iris (in-progress, tech-coded). LIVE = mint (operational)." Phải mất 30 phút, không phải 3 ngày.

P1 · Important Effort · 0.5d
V

Data-viz tokens missing — portal EA mà không có chart scale

NowDESIGN.md §5 mô tả D3 donut, bar, mindmap, roadmap timeline, org chart. UI kit README nói "out of scope — would be added as additional pages". colors_and_type.css không có --chart-1..N sequential / categorical / diverging scales. Khi developer build chart, họ sẽ pick màu ad-hoc và phá color discipline.

FixAdd 3 chart palettes: (1) Sequential cho maturity (mist → iris → iris-deep, 5 steps). (2) Categorical cho TOGAF domain (navy / iris / mint / rose / gold / coral / slate / teal / amber). (3) Diverging cho variance (gap → fg-3 → mint). Vendor 5 chart specimen cards vào preview.

P1 · Important Effort · 1w
VI

Component coverage thấp ở tầng input

NowSystem có KPI card, status pill, domain badge, button, CTA, insight callout. Nhưng không có: text input, textarea, select, checkbox, radio, toggle, search bar, date picker, modal, toast, tooltip, table, empty-state, error-state, loading skeleton. Portal nội bộ thực tế sẽ chạm vào tất cả các thứ này.

FixSpec hoá 10 component nữa với cùng card-DNA: 8 input controls + table + empty/error states. Mỗi cái 1 preview card. Total system size +10 cards, ngang ngửa shadcn nhưng có brand identity.

P1 · Important Effort · 2w
VII

Dark theme chưa được token hoá

NowChỉ có --bg-ink: #0e0f24 cho "hero / footer rare". Không có dark variant cho fg-1/2/3, border, shadow, accent-tint. Phòng họp tối + máy chiếu sẽ rất chói với --bg: #fcfcff.

FixMirror toàn bộ token vào [data-theme="dark"]. Ink-flip, không phải invert — iris stay iris, gold stay gold, chỉ surfaces + ink flip. Đây là 1 ngày work, ROI cao.

P2 · Nice Effort · 1d
VIII

Accessibility tokens missing

NowKhông có token cho focus ring (color, width, offset). Tap target minimum không được codify. Reduced-motion fallback nói có nhưng không enforce qua util class.

FixAdd --focus-ring: 0 0 0 3px rgba(124,108,245,0.4), --tap-min: 44px, util class .no-motion. Document tại README §VISUAL FOUNDATIONS.

P2 · Nice Effort · 0.5d
IX

Foundation gắn quá chặt với narrative "IPO 2030 · 5 chặng"

NowYear-tone mapping (2026=mint, 2027=iris, 2028=rose, 2029=brand-light, 2030=gold) và "5 chặng Roman numeral" sống ở DESIGN.md §1 và §4.4. Nếu strategy shift (e.g. IPO delay sang 2032, hoặc reframe sang AI-First), một nửa system sẽ stale.

FixTách NARRATIVE.md riêng — chứa IPO mapping, year cards, Roman numeral list. Giữ DESIGN.md cho tokens + components không gắn narrative. Khi strategy đổi, chỉ update 1 file.

05 · concrete diffs

Ba thay đổi copy-paste được

Để pha 1 không phải chỉ giấy. Đây là 3 diff thực sẵn sàng apply vào colors_and_type.css.

A · Consolidate font stack

Drop Inter + Dancing Script. Be Vietnam Pro becomes the body workhorse.

/* Was */ --font-body: 'Inter', system-ui, sans-serif; --font-script: 'Dancing Script', cursive; /* Now */ --font-body: 'Be Vietnam Pro', system-ui, sans-serif; // Dancing Script removed — no usage /* Google Fonts import trims to */ family=Montserrat&family=Playfair+Display&family=JetBrains+Mono

B · Add chart palette tokens

One sequential, one categorical, one diverging. Locks data-viz to brand palette.

/* Chart · sequential (maturity) */ --chart-seq-1: #ecedf8; --chart-seq-2: #c5c1f0; --chart-seq-3: #9d94e8; --chart-seq-4: #7c6cf5; --chart-seq-5: #4a4bc8; /* Chart · categorical (TOGAF domain) */ --chart-cat: var(--brand), var(--iris), var(--mint), var(--rose), var(--gold), #fb923c, #64748b, #14b8a6; /* Chart · diverging (variance) */ --chart-div-neg: var(--gap); --chart-div-mid: var(--fg-3); --chart-div-pos: var(--mint);

C · Add a11y + motion tokens

Codify what's currently in prose. Make focus + tap-target enforceable.

/* Accessibility */ --focus-ring: 0 0 0 3px rgba(124,108,245,0.40); --focus-ring-gold: 0 0 0 3px rgba(252,175,22,0.45); --tap-min: 44px; /* Motion choreography */ --motion-enter: 250ms cubic-bezier(.34,1.56,.64,1); --motion-exit: 180ms cubic-bezier(.4,0,.2,1); --motion-hover: 150ms cubic-bezier(.4,0,.2,1); --motion-press: 80ms ease-out;

D · Open dark theme

Mirror surfaces + ink. Accents stay branded.

[data-theme="dark"] { --bg: #0e0f24; /* Ink */ --bg-2: #16175a; --bg-muted: #1f2046; --fg-1: #f4f5fb; /* Mist on dark */ --fg-2: #c5c8e0; --fg-3: #8a8db0; --border: #2a2c5a; --border-light: #1f2046; /* accents pass through unchanged */ }
