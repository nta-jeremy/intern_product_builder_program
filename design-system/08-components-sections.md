## Roman Numeral Section Pattern

Every major section within a page uses Roman numerals I вҶ’ VI (or more):

```
I    вҶ’ Hero / Introduction
II   вҶ’ First content block
III  вҶ’ Second content block
IV   вҶ’ Data / Visualization
V    вҶ’ Analysis / Supporting
VI   вҶ’ CTA / Next chapter
```

**Styling:** Large, faded/muted background number вҖ” decorative function, not navigation.

---

## Chapter Label Pattern

```
ChЖ°ЖЎng [NN] В· [Theme] вҖ” [Title]
```

| Chapter | Theme | Example |
|---------|-------|---------|
| 01 | AI-native Г— Craftsmanship | `ChЖ°ЖЎng 01 В· AI-native Г— Craftsmanship` |
| 02 | Bб»©c tranh | `ChЖ°ЖЎng 02 В· Bб»©c tranh вҖ” Value Stream & Platforms` |
| 03 | Chiбәҝn lЖ°б»Јc | `ChЖ°ЖЎng 03 В· Chiбәҝn lЖ°б»Јc вҖ” Khung sб»‘ hГіa` |
| 04 | HГ nh trГ¬nh | `ChЖ°ЖЎng 04 В· HГ nh trГ¬nh вҖ” Sб»©c khoбә»` |

**Note:** The em dash `вҖ”` separates chapter theme from specific page title.

## Section Pattern Summary

| Pattern | Pages | Purpose |
|---------|-------|---------|
| Roman numeral divider | All | Section counting/navigation |
| Chapter label | All | Portal positioning |

| 5-step roadmap | `/` | Progression visualization |
| Full-bleed timeline bar | `/`, various | Temporal orientation |
| Insight banner/callout | Multiple | Key insight highlight |
| Footnote/note | `/`, `/platforms` | Data source, definition |
| Closing CTA section | All | Drive to next chapter |
